# Kafka SDK

A Spring Boot-based Kafka SDK providing core, common, service, autoconfigure, and starter modules for building Kafka applications.

## Modules

| Module | Description |
|--------|-------------|
| `kafka-sdk-core` | Core Kafka functionality and abstractions |
| `kafka-sdk-common` | Shared utilities and common types |
| `kafka-sdk-service` | Service layer for Kafka operations |
| `kafka-sdk-autoconfigure` | Spring Boot auto-configuration |
| `kafka-sdk-starter` | Starter dependency for quick setup |

## Requirements

- Java 17+
- Maven 3.9+

## Build

```bash
./mvnw clean install
```

## Tech Stack

- Java 17
- Spring Boot 4.1.0
- Spring Cloud 2025.1.2
- Apache Kafka
